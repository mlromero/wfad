jqmListview
directive in module jqm

Description

Creates a jQuery mobile listview. Add jqmLiDivider, jqmLiEntry, and/or jqmLiLinks inside.

Usage

as attribute
<ANY jqm-listview
     [inset="{string}"]
     [shadow="{string}"]
     [shadow="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
inset
(optional)
string

Whether this listview should be inset or not. Default false.

shadow
(optional)
string

Whether this listview should have a shadow or not (only applies if inset). Default true.

shadow
(optional)
string

Whether this listview should have corners or not (only applies if inset). Default true.
